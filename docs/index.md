# Главная страница

Проект создан в рамках учебного задания по разработке и развертыванию статического сайта.

## Изученные технологии

В процессе работы над проектом были изучены следующие инструменты:

- **Git и GitHub** - система контроля версий и удаленный репозиторий
- **Python 3.11** - язык программирования для запуска генератора
- **MkDocs** - генератор статических сайтов на основе Markdown
- **GitHub Actions** - платформа для автоматизации CI/CD процессов

## Использованный стек

В проекте применены следующие технологии:

- Python 3.11.2
- MkDocs с темой Material
- GitHub Pages для хостинга
- GitHub Actions для автоматического деплоя
- Markdown для создания контента

## Структура проекта

Проект включает несколько разделов:
- Главная страница с описанием
- Раздел "О проекте" с технической информацией
- Отчет о выполнении задания
