## 5. Настройка CI/CD

### Создание workflow

Создан файл `.github/workflows/deploy.yml` со следующей конфигурацией:
- Триггер: push в ветку main
- Шаги: установка Python, установка зависимостей, сборка сайта, деплой

### Процесс автоматизации

При каждом изменении в ветке main:
1. GitHub Actions запускает workflow
2. Устанавливается окружение и зависимости
3. Выполняется команда `mkdocs build`
4. Результат публикуется в ветку gh-pages

## 7. Исследование дополнительных вопросов

### 7.1 Российские CDN-провайдеры

Изучены следующие варианты CDN для ускорения доставки контента:

**Yandex Cloud CDN**
- Интеграция с Yandex Object Storage
- Распределенная сеть серверов
- Гибкая тарификация

**Selectel CDN**
- Более 60 точек присутствия в России
- Поддержка HTTPS
- API для управления

**VK Cloud CDN**
- Часть облачной платформы VK Cloud
- Интеграция с другими сервисами платформы

### 7.2 GitVerse для CI/CD

GitVerse - российская платформа для хостинга Git-репозиториев.

Основные возможности:
- Хостинг репозиториев с хранением данных в РФ
- Встроенная система CI/CD через файлы `.gitverse-ci.yml`
- Интеграция с российскими облачными провайдерами
- Техподдержка на русском языке

Пример конфигурации CI/CD:
```yaml
stages:
  - build
  - deploy

build_site:
  stage: build
  script:
    - pip install -r requirements.txt
    - mkdocs build
```

### 7.3 Варианты деплоя статических сайтов

**GitHub Pages**
- Преимущества: бесплатно, простая настройка, SSL
- Недостатки: ограничение на размер, только статика

**Yandex Object Storage**
- Преимущества: масштабируемость, CDN, российская юрисдикция
- Недостатки: платный сервис, требуется настройка

**Собственный VPS**
- Преимущества: полный контроль, гибкость настройки
- Недостатки: требуется администрирование, настройка безопасности

**Netlify / Vercel**
- Преимущества: удобный интерфейс, много функций
- Недостатки: зарубежные сервисы

Необходимые инструменты для деплоя:
- Git для контроля версий
- SSH для доступа к серверу (при использовании VPS)
- Nginx/Apache для веб-сервера
- Certbot для SSL-сертификатов
- AWS CLI или аналоги для работы с Object Storage
